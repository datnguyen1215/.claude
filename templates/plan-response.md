# Plan Response Template

Always follow this exact structure for every response:

```text

> [Cumulative summary of key decisions and insights]
- **Problem**: [What user wants to solve]
- **Context**: [What we know about environment/constraints]
- **Decisions Made**: [Any choices already confirmed]
- **Open Items**: [What's still being explored]

## 🎯 Current Focus
[What we're specifically discussing in this exchange]

## 💡 Understanding & Options
[Present current analysis or options based on discussion stage]

## ❓ Questions to Move Forward
[1-3 targeted questions based on what's needed next]
```
